CoffeeShop Project (English Version)

Welcome to the English version of the CoffeeShop website! This project is a simple, clean, and responsive coffee shop website built with HTML, CSS, and JavaScript.

Features
	•	Responsive design – Works perfectly on desktop, tablet, and mobile devices.
	•	Image carousel – Smooth sliding images to showcase products or promotions.
	•	Multiple pages – Includes Home, Menu, About, Delivery, and Contact pages.
	•	Footer – Consistent footer with contact information and social links.
	•	Clean and optimized images – All images are optimized using TinyPNG for faster loading.

Technologies Used
	•	HTML5 – Structure and content of the website.
	•	CSS3 – Styling and responsive layout.
	•	JavaScript – Interactive features like the image carousel.

How to Use
	1.	Clone the repository:
  git clone https://github.com/Cara1602/Coffeeshopeng.git
  2.	Open index.html in your browser to view the site.
	3.	Edit HTML, CSS, or JS files to customize the website.

Notes
	•	Images are stored in the images/ folder and are optimized for fast loading.
	•	The site is fully responsive; make sure to test it on different screen sizes.
	•	You can extend the carousel by adding more images in the carousel section in HTML.
